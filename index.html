About Me

I'm Swikrit Khatri, a Bachelor of Information Management (BIM) student at Bhaktapur Multiple Campus, Tribhuvan University — building at the intersection of web development and UI/UX design.

I build complete web applications using Next.js, handling both frontend and backend — from responsive interfaces to server-side logic and real data handling. One of my projects is a fully functional food ordering website built end-to-end with Next.js. I also design user-friendly interfaces in Figma — from wireframes and user flows to polished, interactive prototypes.

I believe great digital products need both: code that works and design that feels right. I bring curiosity, attention to detail, and a user-first mindset to everything I build.
